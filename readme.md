## 功能
 weex plugin for vue-cli

## 使用

### 方法一
```
npm i -g @vue/cli
```

```
vue add  vue-cli-plugin-weex 
```

```
npm  run weex:dev
```


### 方法二
```
npm i -g @vue/cli
```

```
vue create -p https://github.com:aymonYU/weex -c  test
```

```
cd test
```

```
npm run weex:dev
```


### 特性
1. 如果使用了@vue/cli-plugin-typescript，则此插件支持typscript

2. 默认入口文件：`./src/main`
